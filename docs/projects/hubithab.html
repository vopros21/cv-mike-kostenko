HubitHab — Automatic Habit Tracker

iOS app · 2024–present · hubithab.eu

Overview

HubitHab tracks habits automatically by pulling data from external services — no manual logging required. It integrates with GitHub commit activity, LeetCode submissions, Apple HealthKit (steps), and X/Twitter posts, then visualises streaks and history as heatmaps.

Tech Stack

Swift / SwiftUI, HealthKit, Spring Boot proxy backend (Java, Render hosting).

Architecture

Connection-based plugin model — each service conforms to a protocol with its own renderer (ConnectionViewFactory). The backend is a thin authenticated proxy that keeps third-party credentials off-device, so the app never stores API keys locally.
